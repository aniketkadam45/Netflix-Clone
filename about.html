About Netflix Clone

A modern Netflix-inspired streaming UI built using HTML, CSS & Bootstrap

What is this project?

This Netflix Clone is a front-end web project designed to replicate the look and feel of the Netflix platform. It focuses on UI/UX design, responsiveness, and clean layout structure.

The project demonstrates the use of Bootstrap components, custom CSS, video backgrounds, and responsive movie cards to deliver a premium streaming-style interface.

NETFLIX

Key Features

Movie & TV Listings

Grid-based layout showcasing movies and TV shows with posters and details.

Responsive Design

Fully responsive across desktops, tablets, and mobile devices.

Video Hero Section

Auto-playing background video with overlay text for a cinematic feel.
